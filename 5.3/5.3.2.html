  • 5.3.2.1. Подготовка отверстий выводов водорозеток.

    Необходимая документация для выполнения работ: Разделы КР.М, ИР.Водоснабжение.

    Производимые операции:

    1. Разметить места расположения выводов воды в модуле.
    2. Просверлить отверстия коронкой/перьевым сверлом.
    3. Снять заусенцы по краям отверстий.

    Контрольные параметры:

    • Проектное положение отверстий: +-1мм.
    • Отстутствие заусенцев на торцах отверстий с лицевой стороны панели.

    Необходимый инструмент и оборудование:

    • Аккумуляторный шуруповерт с зажимным патроном.
    • Коронка.
    • Рулетка 5м.
    • Уголок плотницкий металлический 150мм.
    • Карандаш простой.
    • Нож.

    Расходные материалы:

    • Шкурка, зерно 120.

  • 5.3.2.2. Монтаж водорозеток.

    Необходимая документация для выполнения работ: Разделы КР.М, ИР.Водоснабжение.

    Производимые операции:

    1. Подготвить комплектовку водорозеток согласно спецификации листа проекта.
    2. У каждого отверстия со стороны установки водорозетки (внутри панели стены) закрепить саморезами СЧ 3,5х55 дистанционные проставки толщиной 40мм для крепления водорозетки.
    3. Установить водорозетку в проставку, правильно ориентируя относительно подключения к трассе водоснабжения. Закрепить водорозетку саморезами ПШО 4,2х25. Минимум 2 самореза на каждую водорозетку.

    Контрольные параметры:

    • Соосность водорозеток и просверленных отверстий.
    • Наличие минимум 2-х саморезов в узлах крепления водорозеток.

    Необходимый инструмент и оборудование:

    • Аккумуляторный шуруповерт.
    • Бита PZ2.

    Расходные материалы:

    • Саморезы ПШО 4,2х25.
    • Саморезы СЧ 3,5х55.

  • 5.3.2.3. Прокладка трасс водоснабжения.

    Необходимая документация для выполнения работ: Разделы КР.М, ИР.Водоснабжение.

    Производимые операции:

    1. Подготвить комплектовку труб и фитингов согласно спецификации листа проекта.
    2. Проложить трубы из сшитого полиэтилена согласно проекта. Повороты трасс выполнить при помощи специальных накладок. Трасса к водорозеткам должна идти вертикально. Максимальное отклонение от вертикальной оси водорозетки должно быть не более 100мм.
    3. Выполнить присоединение труб к водорозеткам.
      4. Видеоруководство по подключению труб к водорозеткам.

    Контрольные параметры:

    • Отклонения трассы от проектного положения: +-15мм.
    • Трубы проложены в 1 ряд вдоль стены. Пересечения труб допускаются только при ответвлениях трассы.
    • На сгибах труб установлены специальные фиксаторы.
    • Все узлы водорозеток и фитингов опрессованы.
    • Отсутствие течи и подтеканий при опрессовке.

      • По факту контрольных замеров ставится отметка в паспорте ОТК.

      По итогам законченного этапа сделать фотоотчет выполненных работ.
      На фотоотчете необходимо показать: общий вид трассы, укрупненные фотографии узлов, последовательные укрупненные фотографии трассы, чтобы можно было отследить схему прокладки до каждого потребителя. На каждом фото должна быть табличка с ID объекта, не перекрывающая информацию фотоотчета.

    Необходимый инструмент и оборудование:

    • Пресс для труб из сшитого полиэтилена.
    • Резак для труб.
    • Нож.
    • Рулетка 5м.

    Расходные материалы:

    • Трубы из сшитого полиэтилена.
    • Угловые фиксаторы.
    • Тройники - отводы.

  • 5.3.2.4. Теплоизоляция труб трассы с холодной водой.

    Необходимая документация для выполнения работ: Разделы КР.М, ИР.Водоснабжение.

    Производимые операции:

    1. Поверх труб трассы с холодной водой надеть утеплитель из вспененной полиэтиленовой изоляции (Энергофлекс).

    Контрольные параметры:

    • Не укрытые участки труб: не более 10мм.

    По итогам законченного этапа сделать фотоотчет выполненных работ.
    Общий план стены, на котором видно, что трубы утеплены.

    Необходимый инструмент и оборудование:

    • Рулетка 5м.
    • Нож.

    Расходные материалы:

    • Сменные лезвия для ножа.
