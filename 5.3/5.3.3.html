  • 5.3.3.1. Подготовительные работы.

    Необходимая документация для выполнения работ: Разделы КР.М, ИР.Канализация.

    Производимые операции:

    1. Разметить проектное положение трассы канализации в панелях модуля.
    2. Просверлить коронкой отверстия для трассы.
    3. Подготвить комплектовку канализационных труб согласно спецификации листа проекта.

    Контрольные параметры:

    • Высверлены все отверстия для монтажа трассы.
    • Подготовлена комплектовка материалов по спецификации.

    Необходимый инструмент и оборудование:

    • Аккумуляторный шуруповерт с зажимным патроном.
    • Коронка.
    • Рулетка 5м.
    • Уголок плотницкий металлический 150мм.
    • Карандаш простой.
    • Нож.

    Расходные материалы:

    • Шкурка, зерно 80.

  • 5.3.3.2. Монтаж трассы.

    Необходимая документация для выполнения работ: Разделы КР.М, ИР.Канализация.

    Производимые операции:

    1. Проверить наличие употнительных манжет.
    2. Произвести сборку трассы канализации. В случае, когда требуется подрезка канализацинной трубы в проектный размер, необходимо с укороченной стороны снять фаску и удалить заусенцы.
      Перед соединением обязательно обработать манжеты сантехнической смазкой. Запрещается соединять трубы без фасок и с сухой манжетой!
    3. Зафиксировать трассу перфорированной лентой. Ленту крепить к элементам стеновых панелей саморезами ПШО 4,2х13, обжимая канализационные трубы. В канализационные трубы саморезы вкручивать запрещено.

    Контрольные параметры:

    • Отклонение трассы от проектного положения: +-5мм.
    • Уклон трассы соответствует проектному.
    • Установлены все уплотнительные манжеты.
    • Отсутствует "свободный ход" смещение относительно проектного положения при легком силовом воздействии трассы.

      • По факту контрольных замеров ставится отметка в паспорте ОТК.

      По итогам законченного этапа сделать фотоотчет выполненных работ.
      На фотоотчете необходимо показать: общий вид трассы, укрупненные фотографии узлов, последовательные укрупненные фотографии трассы, чтобы можно было отследить схему прокладки до каждого потребителя. На каждом фото должна быть табличка с ID объекта, не перекрывающая информацию фотоотчета.

    Необходимый инструмент и оборудование:

    • Аккумуляторный шуруповерт.
    • Ножевка по металлу.
    • Бита PZ2.
    • Нож.
    • Ветошь.

    Расходные материалы:

    • Саморезы ПШО 4,2х13.
    • Сменные лезвия для ножа.
    • Сантехническая смазка.
